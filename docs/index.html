Simple rules and complex patterns

I am fascinated by iterative systems that can, despite the simplicity of their definition (I agree this is not objective), engenders quite complex patterns. There are several such systems and I try to list and illustrate the ones I know about in this page. Some of these systems are related to understanding the climate (Lorentz attractor), the formation of patterns on the fur of animals (stripes, spots, ..), the apparent coordinated behaviors of birds, the patterns exhibited by chemical reactions (Belousov-Zhabotinsky reaction), some others are, on first sight, pure mathematical constructions (Mandelbrot set, Julia set).

2.   The chaos game

Back to the top

The chaos game was introduced by Michael Barnsley. It consists in repeatidly applying a function \(f\) starting from a point \(x_0\). This leads to a sequence of points that can, eventually, produce a pattern. The function \(f\) can comprise several deterministic rules that are selected randomly with fixed probabilities.

4.1.   Definition

L-Systems are rewriting systems based on rules. These are symbolic systems, where you define a vocabulary, a set of rules, a stem and then iterate the rewritting rules. A rewritting rule is simply a rule which maps a set of symbols to another set of symbols. One can actually define simple rules that engender complex patterns.
